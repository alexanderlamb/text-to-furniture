# Text-to-Furniture

A custom text-to-flat-pack furniture model and product generator.

## Overview

This project enables the generation of flat-pack furniture designs from text descriptions. The system converts natural language descriptions into parametric furniture models that can be manufactured as flat-pack products.

## Features

- Text-based furniture design specification
- Parametric model generation
- Flat-pack optimization for efficient shipping and assembly
- Assembly instructions generation
- Export to common CAD/manufacturing formats

## Project Structure

text-to-furniture/

├── models/        # Furniture model definitions
├── docs/          # Documentation

└── examples/      # Example designs and outputs
